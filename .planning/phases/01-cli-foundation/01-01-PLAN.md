<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI args → file operations | --cwd and phase number args control which files are read/written |
| JSON parse → state mutation | Malformed STATE.md/ROADMAP.md input could corrupt state |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | findProjectRoot --cwd | mitigate | Validate --cwd with path.resolve(), verify directory exists with fs.existsSync(), never traverse above os.homedir() |
| T-01-02 | Tampering | phase number input | mitigate | Validate phaseNum is numeric before using in path.join(); reject NaN with error message |
| T-01-03 | Information Disclosure | readFile arbitrary path | accept | CLI runs as local user; no privilege escalation. File access bounded by user permissions. |
| T-01-04 | Denial of Service | Large JSON output | mitigate | output() function writes to temp file when JSON > 50KB, preventing stdout buffer overflow |
| T-01-05 | Tampering | YAML frontmatter injection | accept | STATE.md is controlled by the tool itself; no untrusted input enters frontmatter parsing |
</threat_model>
