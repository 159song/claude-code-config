| Config file → init output | Config.json values flow into init output without sanitization |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-01 | Tampering | Phase number in init phase-op | mitigate | Validate phaseNumStr is numeric (parseInt + isNaN check) before passing to findPhaseDir; reject with error message |
| T-02-02 | Information Disclosure | project_root in JSON output | accept | Project root is intended output; CLI runs locally as the user. No privilege escalation possible. |
| T-02-03 | Spoofing | Unknown init sub-mode | mitigate | Whitelist valid sub-modes in run(); reject unknowns with error listing valid options |
