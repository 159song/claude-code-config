# WF 视觉规范

符号: ✅完成 ⬜待处理 🔄进行中 ❌失败 ⚠️警告 ⛔阻塞 ▶下一步

> 上述符号是 WF 系统的视觉设计选择，不受通用 no-emoji 指导约束。

格式规则:
- **横幅:** `╔══════════ WF · {{名称}} ══════════╗` 主要步骤开始时使用
- **检查点框:** `┌─ 标题 ─┐` / `│ 内容 │` / `└───┘` 展示检查点和结果
- **进度条:** `████████░░░░░░░░ 47%` 用 █(完成)/░(未完成) 共16字符，右侧百分比
- **路由:** `▶ 下一步: /wf-xxx` 附原因说明
- **错误:** `⛔ 阻塞: {{原因}}` 附建议操作
- **Agent 启动:** `启动 wf-executor [阶段 N, 计划 X] | 隔离: worktree`
- **验证结果:** 4级表格，每行 `EXISTS/SUBSTANTIVE/WIRED/DATA-FLOWING` 显示 ✅/❌ 计数
